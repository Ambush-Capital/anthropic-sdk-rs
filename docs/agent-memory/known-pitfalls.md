# Known Pitfalls

Recurring mistakes and failure modes. Max 10 entries. Remove least relevant when adding new.

## Pitfalls

### Root cargo commands can miss example breakage

Root `Cargo.toml` sets `default-members = ["anthropic-ai-sdk"]`. After touching shared APIs or example path dependencies, run `cargo check --workspace` or a targeted example crate instead of assuming plain root `cargo check` covered the repo.

### Generated documentation paths look editable but are not source of truth

`documentation/anthropic-sdk-rs/` is generated by Docpup and ignored by git. Make product changes in `examples/` or `anthropic-ai-sdk/`, then regenerate docs only if the local snapshot needs to move.

### API reference questions need the local indices first

Repo instructions require `documentation/library-docs.md` and the index files before training knowledge or ad hoc browsing. Skipping that step is the fastest way to ship stale Anthropic or Claude API assumptions.
